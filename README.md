
Create an approximation of the selected picture using random & selection
as the natural selection do.
 
 usage :   py evoLisa.py -fname ./Monnalisa/Monnalisa_160x200.png

 -fname indicates the path to original Monnalisa picture.

 then all the approximate images creates by evoLisa are inserted
 into the same dir ./Monnalisa/
